##In this task, we created a Python program that automatically generates a report from a data file.

The program first reads data from a file (like a CSV) that contains student names and marks. Then it processes the data to find important details such as:

Average marks
Highest marks
Lowest marks

After analyzing the data, the program creates a neat and organized report that shows all the results in a clear format (like a table or document).

The main goal of this task is to save time by automating the report-making process instead of doing it manually. It helps in understanding how Python can be used for reading files, analyzing data, and generating reports easily.
